# Twig - Framework PHP

Para arquivos com extensão .tpl para representações PHP.

## Exemplos de uso

```{{ product.name }}```

```{% if products | length > 0 %}...{% endif %}```

```{% set teste = "Teste" %}``` > ```{{ teste }}```

```{% include "snipplets/product/product-form" %}```
